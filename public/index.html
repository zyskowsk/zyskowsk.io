about me

I am a mathematician turned web developer living in the bay area. Currently, I work with Backbone.js and Ruby on Rails, but nothing excites me more than learning new technology. I am interested in data analysis and visualization, education, food, and crowd sourcing big ideas. When I am not at my computer you can find me climbing or hiking in the mountains.

Projects

This is a selection of projects I have worked on over the past six months. For code samples and more projects you can visit my github .

Twenty Films

Twenty Films is a social web app where users share a list of their twenty favorite films. This is a single page app built with Backbone.js on top of a Rails API. For this project I focused on aggregating existing data. I built an auto complete search mechanism that gathers film data form multiple databases and complies the results. Other technology used includes Facbook integration, Redis cacheing, JQuery, and JQuery UI. Twenty Films was deployed using Heroku and is live here, source code and demo account information can be found here .

Shared Houses

Shared Houses is a web app I built for housemates who share expenses. Users can add purchases and create monthly statementes that calculate who owes who and how much. I build this app using Ruby on Rails, Devise for authentication and Bootstrap for styling. Shared Houses was deployed using Heroku and is live here, source code and demo account information can be found here .
